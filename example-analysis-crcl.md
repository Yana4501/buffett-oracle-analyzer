## ⚖️ 巴菲特的裁决（3句话执行摘要）

**Circle是一家以美元稳定币USDC为核心的"准银行"——它把用户存入的近800亿美元购买美国短期国债，赚取利息收入，然后与Coinbase等分销伙伴分成。** 2025年营收同比增长64%至27.5亿美元，但公司仍陷于"账面亏损"（净亏7010万美元），因为IPO股权激励费用高达4.24亿美元——剔除此项后经营性盈利能力实际极为强劲。**当前股价约90美元，从最高点298美元跌去近70%，估值从IPO初期的疯狂回归理性，但这个价格是否值得买，取决于你如何看待未来3年USDC流通量能否翻倍、以及利率环境的演变。**

## 模块3：商业模式深度解析

### 一句话测试（巴菲特测试）

> **Circle是一家"代币化货币基金"——你存入1美元，它去买国债，你得到1个USDC，它赚取国债利息，然后与Coinbase等分销商分成。**

### 收入结构（2025全年）

| 收入来源 | 金额 | 占比 | 驱动因素 |
|----------|------|------|----------|
| **储备收入（Reserve Revenue）** | $26.37亿 | 96% | USDC流通量 × 储备收益率 |
| **其他收入** | $1.10亿 | 4% | 支付服务、订阅服务、API收入 |
| **合计** | $27.47亿 | 100% | |

### USDC经济学（核心商业模式）

```
USDC发行 → 用户存入美元 → Circle购买短期美国国债
         → 国债利息收入（储备收益率）
         → 减去分销成本（给Coinbase等）
         → = Circle的净收入
```

### USDC流通量（截至2026年3月）

| 指标 | 数值 |
|------|------|
| 最新流通量 | 约793亿美元（历史新高） |
| 年增长 | +72%（从438亿美元增长） |
| 链上交易量 | 已超越USDT（2026年3月首次） |

### 关键成本——"给Coinbase打工"

- Circle将约50%的储备收益支付给Coinbase
- 2025全年分销成本高达10亿美元以上
- 即便储备收入可观，净利润率仍受压

### 产业链定位

```
上游：持有USDC的用户/机构
      ↓
中游：Circle（发行商，管理储备）
      ↓
下游：Coinbase等交易所/钱包（分销渠道）
      国债/现金（储备资产）
```
