## 这是什么？

**巴菲特神谕分析师**是一个 [Claude AI Skill](https://docs.anthropic.com)，能够对任何上市公司（美股、港股、A股）进行全面的、机构级别的深度分析。

它将沃伦·巴菲特60多年致股东信中的投资智慧，与一套经过实战验证的**12模块、50+细分维度**分析框架相融合。

给它一个公司名称或股票代码，它将输出：

| 能力 | 细节 |
|------|------|
| 📊 **12大分析模块** | 从商业模式到K线技术形态，全方位覆盖 |
| 🏰 **护城河5维度评分** | 网络效应、转换成本、成本优势、无形资产、有效规模 |
| 💰 **所有者盈余计算** | 巴菲特最推崇的盈利指标 |
| 🧮 **8种估值模型** | DCF、相对估值、PEG、DDM、NAV、SOTP、所有者盈余收益率、反向DCF |
| 📞 **电话会深度解读** | 最新一期 + 前两期对比，含管理层措辞变化追踪 |
| 📈 **技术面完整分析** | 均线系统、RSI/MACD/布林带、斐波那契、筹码分析 |
| 🎯 **巴菲特计分卡** | 12项标准、36分满分评分体系 |
| 🔴🟡🟢 **可执行决策** | 目标价(三情景)、买入区间、止盈/止损、仓位建议、分批建仓策略 |

## 6步分析工作流

| 步骤 | 环节 | 动作 |
|------|------|------|
| 1 | 🔍 信息采集 | 搜索最新财报、电话会、新闻、研报、内部人交易 |
| 2 | 📋 基本面分析 | 业务+财务+管理层+竞争全面拆解 |
| 3 | 🧮 估值建模 | 8种估值模型交叉验证合理价值区间 |
| 4 | 📉 技术面研判 | K线形态+指标+支撑阻力位+筹码分析 |
| 5 | ⚠️ 风险评估 | 多维风险矩阵+黑天鹅压力测试 |
| 6 | 📄 输出报告 | 巴菲特计分卡+综合评级+目标价+操作建议 |

## 12大分析模块

| # | 模块 | 覆盖内容 |
|---|------|---------|
| 1 | **公司概览** | 背景、里程碑、股权结构、公司治理 |
| 2 | **管理层评估** | 领导力、资本配置、利益绑定、巴菲特主人翁测试 |
| 3 | **商业模式** | 收入模型、单位经济、一句话测试、产业链定位 |
| 4 | **经济护城河** | 5维度评分 + 趋势(加宽/稳定/侵蚀) |
| 5 | **财务深度** | 4季度趋势、三张表、杜邦、资产负债健康度、现金流质量、所有者盈余 |
| 6 | **电话会分析** | 最新+前2期对比、指引追踪、措辞变化 |
| 7 | **多模型估值** | DCF/相对/PEG/DDM/NAV/SOTP/所有者盈余收益率/反向DCF（共8种） |
| 8 | **竞争与行业** | 份额、波特五力、差异化优势、颠覆风险 |
| 9 | **技术面分析** | 均线(MA5-250)、RSI/MACD/布林带、斐波那契、筹码分析 |
| 10 | **增长与催化剂** | 驱动力、TAM、短期/长期催化剂 |
| 11 | **风险矩阵** | 6维度评分(1-5) + 黑天鹅情景 |
| 12 | **巴菲特计分卡** | 12项/36分评分 + 评级 + 目标价 + 买卖策略 + 仓位管理 |

## 巴菲特计分卡

| 标准 | 满分 | 巴菲特关注的是什么 |
|------|------|------------------|
| 业务是否易于理解 | 3 | 10岁的孩子能理解吗？ |
| 经营历史是否一致 | 3 | 10年以上稳定增长的盈利记录 |
| 长期前景是否良好 | 3 | 20年后需求还会存在吗？ |
| 管理层坦诚与理性 | 3 | 他们会公开承认错误吗？ |
| 主人翁经营心态 | 3 | 是否有大量个人持股？ |
| 高ROE低负债 | 3 | >15% ROE且不靠杠杆 |
| 强劲自由现金流 | 3 | FCF持续超过净利润 |
| 宽广经济护城河 | 3 | 持久的竞争优势 |
| 保守会计处理 | 3 | 没有激进的收入确认 |
| 存在安全边际 | 3 | 当前价格低于内在价值 |
| 定价能力 | 3 | 能涨价而不丢客户 |
| 低资本再投入需求 | 3 | 不需要大量投入就能增长 |

**评级**：A（30-36）杰出 · B（24-29）优秀 · C（18-23）一般 · D（12-17）担忧 · F（<12）回避
